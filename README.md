# list

List processing sandbox.

## Prerequisites

- [sbcl](https://www.sbcl.org)
  ```sh
  scoop install main/sbcl
  ```
- [quicklisp](https://www.quicklisp.org/beta/)
  ```sh
  curl -O https://beta.quicklisp.org/quicklisp.lisp
  sbcl --load quicklisp.lisp
  ```
